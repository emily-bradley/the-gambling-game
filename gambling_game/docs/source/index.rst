Welcome to The Gambling Game's documentation!
=============================================

The Gambling Game is a hands-on learning exercise for the Data Science Community of Practice.
It is designed to foster collaborative exploration of the data science topic: Reinforcement Learning.
To participate in the event, follow the steps under :ref:`How Submit an Agent`.

If you want to learn more about the Gambling Game, read on. If you are interested in getting started, check out the :doc:`getting started <getting_started>` section for further information.
The game framework details are available under :doc:`usage <usage>`.
